# 📊 CRM Data Analysis & Cleansing

## Sobre o Projeto
Análise exploratória e tratamento de uma base de dados bruta simulando o ambiente de um CRM corporativo. O objetivo do projeto foi atuar na limpeza de inconsistências (handling de dados nulos e duplicatas) e estruturar a base para extração de inteligência comercial.

## O Desafio de Negócio
Bases de CRM frequentemente acumulam dados desestruturados devido à inserção manual, o que impossibilita a leitura clara do funil de vendas e do comportamento dos leads. O foco foi transformar "dados sujos" em um dataset confiável para consumo de dashboards e relatórios gerenciais.

## Processo e Transformação
1. **Limpeza e Padronização:** Identificação e tratamento de valores ausentes (Missing Values), remoção de duplicatas e padronização de tipagem de dados (datas, moedas, strings).
2. **Análise Exploratória (EDA):** Extração de métricas descritivas para entender o perfil dos clientes e a distribuição das vendas.
3. **Consolidação:** Geração de uma base tratada, pronta para ser conectada a ferramentas de visualização de dados (Business Intelligence).
